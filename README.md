# myInvestment — 個人資產配置儀表板

把你的**海外基金、台股、美元存款、台幣存款**集中在一頁，全部換算成台幣，
用圓餅圖看「股債 / 類別 / 幣別 / 地區」配置，依目標比例給**再平衡建議**，並監控**停利**。

## 追蹤模型：B 累計投入式
- **市值** = 持有單位 × 現價（台股自動抓、基金手動淨值、現金=餘額）
- **累計投入** = 投入基準 + 定期定額自動累加（每月 `扣款日` 自動加 `扣款金額+手續費`）
- **損益** = 市值 − 累計投入；報酬率 = 損益 / 累計投入
- 「既往不究」：投入基準設成銀行目前的「投資金額」，損益從今天起算。
- 改動「投入基準」時，基準日自動設為今天，定期定額從此重新累加（不會重複計）。

## 功能
- 多幣別統一換算 TWD；台股價（TWSE MIS）、USD/TWD 匯率（open.er-api.com）自動抓
- 四維度圓餅圖：**股債屬性** / 類別 / 幣別 / 地區
- 定期定額自動累加投入（只需偶爾更新單位 / 淨值）
- 停利監控：每檔可設目標報酬率，達標/接近時於儀表板紅黃旗提醒
- 再平衡對照表（可對股債、類別、幣別、地區設目標）

## 升級（已部署 v1 者）
phpMyAdmin 對 `zjqafhmy_myinvest` 匯入 `db/migrate_b.sql`（只跑一次），再覆蓋上傳程式檔。
既有部位的 `quantity` 即持有單位；現金金額會搬到 `balance`。之後逐筆「編輯」設好屬性、累計投入、定期定額、停利目標。

## 全新安裝（本機）
需要 PHP 8+ 與 MySQL。
```bash
cp config.sample.php config.php
mysql -u root -p myinvestment < db/schema.sql
php -S localhost:8000
```

## 資料怎麼填
| 資產 | 類別 | 屬性 | 怎麼填 |
|------|------|------|--------|
| 台股（一般股票/ETF） | 台股 | 股票 | 代號+股數，勾自動抓價，填累計投入(持有成本) |
| 債券型 ETF（如 00679B） | 台股 | **債券** | 同上，地區設**美國**（投資地，非掛牌地） |
| 海外股票基金 | 海外基金 | 股票 | 單位數+淨值，設累計投入；定期定額勾起來填扣款額/手續費/扣款日 |
| 債券/月配基金（如 NN79） | 海外基金 | **債券** | 同上 |
| 美元 / 台幣存款 | 現金/存款 | 現金 | 幣別+餘額 |

## 之後可加

- 每日總值快照 → 資產成長趨勢圖
- 停利達標每日排程 + 推播通知
